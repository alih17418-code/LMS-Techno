Apply authorization on both frontend and backend.

Do not rely only on UI restrictions.

Full access

Staff:

Operational access only
No delete permissions
No financial reports

Instructor:

Access only assigned classes
Access only assigned students
Access only assigned attendance records
Access only assigned financial data
Access only personal earnings information

All APIs, database queries, dashboards, menus, reports, and actions must strictly follow role permissions.
